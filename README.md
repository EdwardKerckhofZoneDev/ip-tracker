# vue3-ip-tracker

```
Simple IP-tracking Electron app
```

### Technologies used

- Vue 3
- Comosition API
- Electron
- TailwindCSS
- Pinia (Vuex alternative for state management)
- Google Maps Javascript API
- Ipify API

### Install

```bash
git clone https://github.com/EdwardKerckhofZoneDev/ip-tracker.git
cd ip-tracker
yarn install
yarn dev
```
